Instructions for utilizing the various 3d printers in the engineering lab. Under construction.

Ultimaker 3

  1. Create model in your favorite CAD software, then save as .stl
  2. Open .stl file in Cura Ultimaker (should be on school computers)
  3. Orient model for optimal print position (on a flat edge with only a small amount of overhangs(parts of model that does not have any supporting material below it and hangs in the air, if part has many long overhangs then use resin printer.)) Buttons on side allow you to orient part.
  4. Ensure that the correct printer core and material is selected in upper right corner. Usually, the AA core is core 1 and BB core is core 2. As for material, think about what material you need to use (ABS is usually best) and select it from dropdown menu.
  5. Connect USB drive to computer, and click on dropdown in lower right corner to save to usb drive. Click 'slice' in lower right corner to prepare the part for printing and save to drive.
  6. Remove drive from computer and insert into Ultimaker 3.
  7. Check materials spool to make sure it is the type you selected when preparing the print. Click here to remove/change material for the ultimaker.
  8. Click the first option on the printer to open the drive. Select your print (should be the first option.) Click print.
  9. Wait for extruder and bed to head up, then watch print as it starts to make sure it is printing as expected. Do not use adhesives or tape. If unconfident in whether a part of the print will print correctly, please make plans to watch print for that section.
  10. When print is finished, ideally remove print while the bed is still hot but it can be removed when it is cold, too. Remove using prying tools such as the various knives in the printing area. Do not damage the bed. Clean bed affter if neccesary.
